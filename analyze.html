This is an AI-power Sentiment Analysis Tool for Poetry, a part of the Reading between the Lines data visualization project by C.Wu.

You may edit and re-analyze your poem at any time; unfortunately, the sentiment analyzer can only interpret English at the moment.

How the Poetry Sentiment Analyzer Works:

You can use this tool to apply the sentiment analysis techniques covered in Reading between the Lines and analyze any poetry you are interested in, or rather, a poem of your own creation! The analyzer will return a static visual overview of your poem. To start the analyzer, type in the poem of your choice in the input box on the left, and click Analyze.
